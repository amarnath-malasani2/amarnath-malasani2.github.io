# amarnath-malasani2.github.io
My personal portfolio website showcasing projects, skills, and experience in DevOps and software development.

## Features
- Responsive design that works on mobile, tablet, and desktop
- Sections for showcasing skills, projects, and work experience
- Document management system (upload, view, download, delete)
- Note-taking functionality
- Firebase integration for backend storage and authentication

## Setup Instructions
1. Clone this repository
2. Update the Firebase configuration in `js/firebase-config.js` with your own Firebase project details
3. Enable Firebase Authentication, Storage, and Firestore in your Firebase project
4. Deploy to GitHub Pages or any static hosting service

## Document Management
This portfolio includes a document manager that allows:
- Uploading documents
- Viewing documents
- Downloading documents
- Deleting documents

## Notes System
Take and manage notes with:
- Create new notes with title and content
- Edit existing notes
- Delete notes
- All notes are saved in Firebase Firestore

## Technologies Used
- HTML5, CSS3, JavaScript
- Firebase (Authentication, Storage, Firestore)
- Font Awesome for icons
- Responsive design for all device sizes
